# PDF Pricing Tool (Standalone)

Standalone app for PDF page pricing analysis:

- Total page count
- Page size per page (points + inches)
- Estimated `B/W` vs `Color` page classification for click charges
- Drag-and-drop PDF upload

## Run locally

```bash
npm install
npm run dev
```

## Build for hosting

```bash
npm run build
```

Deploy the generated `dist/` folder to your temporary hosting target.
